3 Cara Menampilkan Data dalam JavaScript

  1. Window Alert

    Metode window.alert() digunakan untuk memberikan peringatan kepada pengguna melalui jendela pop-up. Ketika window.alert("pesan") digunakan, sebuah jendela pop-up akan muncul di layar pengguna dengan pesan yang ditentukan. Metode ini umumnya digunakan untuk memberikan informasi penting kepada pengguna atau meminta konfirmasi sebelum melanjutkan tindakan tertentu. Meskipun sederhana, peringatan ini dapat menarik perhatian pengguna secara langsung, terutama dalam situasi yang membutuhkan perhatian segera.

  2. Document Write

    Metode document.write() adalah cara lain untuk menampilkan data dalam JavaScript dengan menambahkannya langsung ke dalam dokumen HTML. Dengan menggunakan document.write("teks"), teks atau kode HTML bisa ditambahkan secara langsung ke dalam halaman web. Namun, perlu diperhatikan bahwa penggunaan document.write() setelah halaman dimuat dapat menyebabkan penghapusan konten yang sudah ada di halaman, sehingga sebaiknya digunakan dengan hati-hati agar tidak merusak struktur dokumen.

  3. Inner HTML

    Properti innerHTML adalah properti dari elemen HTML yang memungkinkan pengguna untuk mengubah atau menambahkan konten HTML ke dalam elemen tersebut. Dengan menggunakan innerHTML, pengguna dapat memanipulasi isi dari sebuah elemen HTML atau bahkan menambahkan elemen HTML baru ke dalamnya. Metode ini sering digunakan untuk memperbarui tampilan halaman web secara dinamis berdasarkan interaksi pengguna atau data yang diperoleh dari sumber eksternal, memberikan fleksibilitas dalam mengatur tampilan halaman web sesuai kebutuhan aplikasi. Dengan menggunakan innerHTML, konten yang ada dalam elemen bisa diganti dengan konten baru, atau bahkan elemen baru bisa ditambahkan ke dalamnya secara dinamis.
